# Blinkit Sales Dashboard

An interactive sales analytics dashboard created using Microsoft Excel to analyze Blinkit grocery sales performance, outlet trends, customer ratings, and product category insights.

### Project Overview

This dashboard provides detailed insights into:
- Total Sales Performance
- Average Sales
- Number of Items Sold
- Customer Ratings
- Outlet Establishment Trends
- Outlet Size & Location Analysis
- Item Type Performance
- Fat Content Analysis
---
### Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design
---
### Key Metrics

- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8523
- Average Rating: 4.0
---
### Dashboard Features

- Interactive filter panel
- Outlet size analysis
- Location-wise sales breakdown
- Item category performance tracking
- Fat content comparison
- Outlet establishment trend analysis
---
### Insights Generated

- Tier 3 outlets generated the highest sales.
- Regular fat products contributed more sales compared to low fat products.
- Fruits & Vegetables and Snack Foods were top-performing categories.
- Medium-sized outlets contributed the highest revenue share.

- Dynamic KPI cards
