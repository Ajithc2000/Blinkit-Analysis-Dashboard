<h1>Blinkit Sales Dashboard</h1>

<p>An interactive sales analytics dashboard created using Microsoft Excel to analyze Blinkit grocery sales performance, outlet trends, customer ratings, and product category insights.
</p>

<h3> Project Overview</h3>

This dashboard provides detailed insights into:

- Total Sales Performance

- Average Sales

- Number of Items Sold

- Customer Ratings

- Outlet Establishment Trends

- Outlet Size & Location Analysis

- Item Type Performance

- Fat Content Analysis

<h3>Tools & Technologies</h3>

- Microsoft Excel
  
- Pivot Tables
  
- Pivot Charts

- Slicers
  
- Conditional Formatting
  
- Dashboard Design

<h3>Key Metrics</h3>

- Total Sales: $1.20M

- Average Sales: $141

- Number of Items: 8523

- Average Rating: 4.0

<h3>Dashboard Features</h3>

- Interactive filter panel

- Outlet size analysis

- Location-wise sales breakdown

- Item category performance tracking

- Fat content comparison

- Outlet establishment trend analysis

- Dynamic KPI cards
